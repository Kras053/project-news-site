Time to beat this record - Most matchsticks stacked into a tower in one minute!

The most matchsticks stacked into a tower in one minute is 74, achieved by Silvio Sabba (Italy), in Pioltello, Italy, on 2 July 2012. This record was broken on the Guinness World Record Challengers site.

Read all about this on Guiness World Record site!
